## 🚀 Description

**ShareCloudy Detector** est une extension de navigateur qui détecte automatiquement si une vidéo en cours de lecture est hébergée sur **ShareCloudy**.

Lorsqu’un lecteur compatible est identifié, une **popup discrète** apparaît en bas à droite de l’écran pour vous permettre :

- 📋 de copier le lien de la vidéo  
- 🔗 d’ouvrir directement la source  

---

## ⚙️ Fonctionnalités

- 🔍 Détection automatique des lecteurs ShareCloudy  
- 💡 Interface simple, rapide et non intrusive  
- 📋 Copie du lien en un clic  
- 🔗 Ouverture directe de la source  
- ⚡ Léger et performant

## 🛠️ Installation

### 🔹 Chrome / Brave / Edge (Chromium)

1. Clone le projet :
```bash
git clone https://github.com/C4P1T4IN3/ShareCloudy-Detector.git
```

2. Ouvre ton navigateur et va sur :
```
chrome://extensions/
```

3. Active le **mode développeur**

4. Clique sur **"Charger l’extension non empaquetée"**

5. Sélectionne le dossier du projet

---

### 🔹 Firefox

1. Va sur :
```
about:debugging#/runtime/this-firefox
```

2. Clique sur **"Charger un module complémentaire temporaire"**

3. Sélectionne le fichier `manifest.json`
